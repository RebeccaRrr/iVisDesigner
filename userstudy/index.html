iVisDesigner User Study

Welcome to the iVisDesigner User Study. Please go though the following steps.

Introduction

iVisDesigner is a web-based software that allows you to design information visualizations interactively, without any programming. We are conducting this user study to evaluate the system. Thanks for your participation, and your feedback is extremely useful to us.

Tutorial

Go through the video, and try to understand how the system works and how to use it.

Practice

Please follow the video to create the following visualization designs:

  1. Basic scatterplot for the Cars dataset.
  2. Graph visualization.

Exercise on Free Visualization Design

There are three datasets provided: Cars, Co-occurrence Graph, and Beijing Air Visualization. You are invited to design some visualizations to try our system. Please save your visualization designs.
